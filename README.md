# Sales-Performance-Evaluation-Dashboard-Power-BI
Sales Performance Analysis using Power BI with data modeling, DAX, and Power Query. Built interactive dashboard to analyze revenue, win rate, quarterly trends, and sales performance. Demonstrates skills in data analytics, business intelligence, KPI design, and data visualization for decision-making.

## Project Overview
This project analyzes sales performance using Power BI to track revenue, win rate, and sales resource efficiency.
The dashboard provides interactive insights to help stakeholders monitor KPIs, identify trends, and improve decision-making.

## 🎯 Objectives

Analyze overall sales performance
Track signing value and win rate
Identify top-performing sales resources
Detect quarterly trends in revenue
Compare category distribution (RFC vs RFP)
Build an interactive dashboard for decision-making


## Tech Stack

| Tool          | Usage                          |
| ------------- | ------------------------------ |
| Power BI      | Dashboard & Visualization      |
| Power Query   | Data Cleaning & Transformation |
| DAX           | KPI Measures & Calculations    |
| Data Modeling | Table relationships            |
| Sharepoint   | Data Source                    |


## Dashboard Features

KPI Cards (Total Signing, Avg Signing, Won Leads, Win Rate)
Sales Performance Matrix
Quarterly Signing Trend Analysis
Category Distribution (RFC / RFP)
Resource Performance Ranking
Region & Year Filters for drill-down analysis


## Key Insights

Identified top performing sales resources
Detected decline in signing value in specific quarters
Found differences in win rate across categories
Highlighted performance gaps across regions
Reduced manual reporting by creating automated dashboard


## Business Impact

Improved visibility of sales KPIs
Enabled faster decision-making
Identified performance gaps
Provided single source of truth
Helped management monitor sales efficiency
